# Trabajo Práctico: Fundamentos de TypeScript

Este repositorio contiene la resolución del Trabajo Práctico sobre los conceptos básicos de TypeScript.

## Temas abarcados

- **Tipos de datos:** Primitivos, Uniones y Literales.
- **Estructuras:** Interfaces y Enums.
- **Funciones:** Tipado de argumentos y retorno.
- **Glosario:** Definiciones técnicas.

## Cómo ejecutar el proyecto

1.  **Instalar dependencias:**

    ```
    npm install
    ```

2.  **Compilar a JavaScript:**

    ```
    npx tsc
    ```

3.  **Ejecutar:**
    ```
    node dist/nombre_de_tu_archivo.js
    ```
